# Security Policy

## Unterstützte Versionen

Aktuell wird nur die Hauptversion (Main Branch) aktiv mit Sicherheits-Updates versorgt.

| Version | Unterstützt |
| ------- | ----------- |
| 0.1.x | ✅ Ja |
| < 0.1.0 | ❌ Nein |
