# KGA-Einfahrts-Manager 🚗🌳

Der **KGA-Einfahrts-Manager** ist eine hochmoderne, proprietäre Webapplikation zur digitalen Verwaltung, Prüfung und Abrechnung von Ausnahmegenehmigungen zum Befahren von Kleingartenanlagen (KGA).

Entwickelt unter modernsten Clean-Code-Standards (PHP 8.3+, Domain-Driven Design, Action-Domain-Responder, strict typing), bietet das System höchste Ausfallsicherheit, strikten Datenschutz (DSGVO-konforme Lebenszyklen) und eine intuitive Benutzerführung für Pächter und den Vorstand.

Die vollständige technische Dokumentation der Architektur befindet sich im Ordner `docs/`.
